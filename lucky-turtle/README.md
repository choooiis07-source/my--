# 행운의 거북이

어떤 프로그램을 사용하든 화면 위를 산책하는 Windows 데스크톱 친구입니다.

## 현재 MVP

- 투명하고 항상 위에 표시되는 오버레이
- 일반 화면 영역은 마우스 클릭 통과
- 연결된 모니터와 각 모니터의 작업 영역 감지
- 모니터 가장자리를 따라 자동 산책
- 거북이 이름, 등딱지 색, 산책 속도 설정
- 터치 반응과 반복 터치 시 등딱지에 숨는 모션
- Windows 로그인 시 자동 실행 선택

## 실행

```powershell
npm install
npm run dev
```

프로덕션 리소스를 확인하려면 다음 명령을 사용합니다.

```powershell
npm run build
npm start
```

## Windows 설치 파일 만들기

```powershell
npm run dist:win
```

결과는 `release/Lucky-Turtle-Setup-0.1.0.exe`에 생성됩니다. 공개 배포 전에는 Windows 코드 서명 인증서를 연결해야 SmartScreen 경고를 줄일 수 있습니다.

거북이에 마우스를 올리면 우측에 설정 버튼이 나타납니다.

## 다음 단계

현재는 운영체제가 제공하는 실제 모니터·작업 영역 경계를 길로 사용합니다. 화면 속 UI의 가로선과 세로선을 직접 타게 하려면 화면 캡처 권한, 주기적인 저해상도 캡처, 선 검출을 추가해야 합니다. 이 기능은 개인 정보 보호를 위해 명시적으로 켜는 선택 기능으로 구현하는 것이 좋습니다.
